Part 1 - Webpage Design

My website will be twofold; the first is a step-by-step course walkthrough, with each assignment and unit posted. It will also have information and media about Golf. The description of the pages is as follows:

Page 1 — The index and landing page are outlined in the mockup. They will include the text and information pertaining to this course. Each assignment will be posted within the correct section.
Page 2 — Image page: this will include a collage of images from different golf courses I have played at.
Page 3 — Multimedia Page: This will have videos about golfing and tips on improving your game and swing by some of my favorite YouTubers.
Page 4 — Links Page: This will include links to different websites related to golf. It will include some of my favorite courses and links to websites for purchasing golf equipment.
Page 5 — User Design page. This will have a link to the layout mockup.

My next major step is creating a CSS build for my page to include my chosen color pallet and the “unit in a box” feel I want. I want the landing page to feel like a bulletin board, with each unit and section in an individual box. I will continue to build the page based on the requirements of the course, adding a section for each unit as it progresses. I also plan to make each form an individual form so that the progress of forms can be viewed while exploring the page. I may add screenshots at each breaking point to show the site's development over time.
