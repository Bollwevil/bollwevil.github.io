Unit 5

Part 1 - Using a Form Creation Website

I ended up using JotForm as my form creation website. The user interface was straightforward, and I could create a survey for my rental club business within minutes. The process of creating a form with a pre-made tool is a breeze. I prefer coding my forms independently, as I feel I have ultimate control over the situation. I couldn't match the CSS and style of my page with any form creation tools, not without completely writing the CSS (and half the other code) on my own. However, I can see the appeal for someone who doesn't have experience in writing HTML or CSS. Creating a form with these tools is easy and can significantly increase a website's interactive features without much effort
